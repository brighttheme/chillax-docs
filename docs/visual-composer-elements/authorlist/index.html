Author List

Table of contents
  1. Author List - General
  2. Author List - Design

Author List - General

  1. Total Number of Authors to show - Default is 6. To show all users enter “-1” (all users).
  2. Number of Columns - Total number of columns in a row.
  3. Select Single/Multi Author Role In - Select the single or multipule Author Roles. Note:- To Select multiple items:- Press and hold the Control(windows)/Command(mac) key, then click the items.
  4. Authors include - User IDs to include (comma separated). Example:- 1,2,3
  5. Authors exclude - User IDs to exclude (comma separated). Example:- 1,2,3
  6. Order by - Sort retrieved posts by selected parameter.
  7. Order - Designates the ascending or descending order of the ‘Order by‘ parameter. Defaults is descending.
  8. Show Author Description - Show or hide author description.

Author List - Design

  1. Image Style - Select the author image style.
  2. CSS box - Author item block styles.
