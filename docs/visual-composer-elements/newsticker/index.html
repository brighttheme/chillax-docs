News Ticker - General

  1. Number of Posts To Show - Total Number of Posts to Show. Default value is 5.
  2. Ticker Title - Customize News Ticker Title, leave blank for hide title lable.
  3. Animation/Effect - Select sliding animation.
  4. Direction - LTR(left-to-right) languages display content from left to right. RTL(right-to-left) languages display content from right to left.
  5. Autoplay - Enable or disable the slide autoplay.
  6. Delay timer - The amount of time in milliseconds.
  7. Scroll Speed - The amount of time in milliseconds.
  8. Stop On Hover - Sliding will pause on hover.
  9. Position - News Ticker dispaly position.
  10. Enable Slider Indicators(next/prev) button - Enable Slider Arrow for navigation.
  11. Enable slider play/pause button - Enable Slider play/pause button.
  12. Show Category? - Show post category.
  13. Show publishing time - Show post publishing date.
  14. Show News Number? - Show post number.

News Ticker- Data Filter

  1. Offset - Number of post to displace or pass over. If you want to skip the 5 posts and start with 6’th post on this block, then simply enter 5.
  2. Include Category - Select Single or Multiple Category to show posts. Note:- To Select multiple items, Press and hold the Control(windows)/Command(mac) key, then click the items.
  3. Show Posts From Tags - Display posts that have “either” of these tags:(Example:: bread,baking). Display posts that have “all” of these tags:- (Example:: bread+baking+recipe).
  4. Include/Exclude Author by ID - Display posts from several specific authors (Include Example:- 2,6,17,38). Display all posts except those from an author(singular) by prefixing its id with a ‘-‘ (minus) sign. (Exclude Example:: -12)
  5. Include posts by Post IDs - Display posts by Post IDs ( Example:- 2,6,17,38 )
  6. Exclude posts by Post IDs - Display posts by Post IDs ( Example:- 2,6,17,38 )
  7. Ignore Sticky Posts - Ignore the procedure of sticky posts.
  8. Order by - Sort retrieved posts by selected parameter.
  9. Order - Designates the ascending or descending order of the ‘Order by‘ parameter. Defaults is descending.

News Ticker - Design

  1. Show Title Blinker? - Blinker icon animation.
  2. Height (px) - Height of the News Ticker.
  3. Font-Size - Post title text font size.
  4. Theme Color - News Ticker theme color.
  5. Title Color - News Ticker title color.
  6. Text Color - News Ticker post title color.
  7. Background Color - News Ticker background color.
  8. Border Width (px) - News Ticker block border color.
  9. Border Radius (px) - News Ticker block border redious.
  10. Z-Index - News Ticker Z-Index.
  11. Block Shadow - News Ticker block box shadow.
